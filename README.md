# TokiToki Android App

## Branch 管理

1. MASTER  
リリースするためのブランチ

2. DEVELOP  
MASTERに入れる前のブランチ

3. DEVELOP_/x.x.x  
特定バージョンの開発するためのブランチ、DEVELOPブランチから切る  

4. FEATURE_/issue_#XXXX  
DEVELOP_/X.X.Xブランチから切る、issueタスク単位のブランチ
